SUPER VELOCITY CURVE — MAC INSTALL
=================================

1. Double-click "Install Super Velocity Curve"
2. If Mac blocks it: System Settings → Privacy & Security → Open Anyway, then run the installer again
3. Quit your music app completely, then open it again and rescan plug-ins

Logic Pro: Super Velocity Curve MIDI FX → track MIDI FX slot
Ableton: Super Velocity Curve → Instrument on a MIDI track

Full guide: https://github.com/azazhang/SuperVelocityCurve/blob/main/docs/user/install.md
