# Naming

Two forms. Everything else is a typo.

| Form | Use for |
|------|---------|
| **Super Velocity Curve** | People — README, docs, plugin window, `PRODUCT_NAME`, `COMPANY_NAME` |
| **SuperVelocityCurve** | Machines — GitHub repo, CMake/`project()`, C++ types, release zips (`SuperVelocityCurve-macOS-unsigned.zip`), bundle IDs (`com.supervelocitycurve.*`) |

Spaces vs no spaces is the only intentional difference. **Do not use** `Super VelocityCurve` (hybrid), `Super_VelocityCurve` (legacy repo), or `Super-Velocity-Curve` (hyphenated — not the same string).

ValueTree XML types (`SuperVelocityCurveState`, etc.) and plugin codes (`Svcv`, `Svc2`) stay stable for saved projects.
